# Screwdriver

## English

Update OBS text with input comes from Youtube channel moderators.

### Steps

1. Install Python >= 3, pip should come with it. Run

```bash
pip install -r requirements.txt
```

2. Enable OBS Websocket Server
3. Go live.
4. Start script.
5. Give required inputs.
6. Done!
